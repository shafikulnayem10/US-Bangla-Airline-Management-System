US-Bangla Airline Management System
Overview

A Java-based airline management system for handling flight bookings, cancellations, user management, and support requests.

How to Run this project:

1)Clone the repo

git clone https://github.com/shafikulnayem10/US-Bangla-Airline-Management-System.git


2)Open in an IDE (NetBeans, IntelliJ, Eclipse).

Build/Compile the project (the IDE will detect the Ant build file).

3)Run  project

->Requirements

1)Java JDK 8+

2)An IDE (NetBeans recommended)
